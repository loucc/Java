# Java 25 基础语法实践建议

> 本文在学完本目录的可运行示例后阅读。重点不是追求最短代码，而是让类型、边界和意图清楚。

## 1. `var`：根据可读性选择

`var` 只能用于有初始化器的局部变量，类型仍在编译期确定，不是动态类型。

```java
var names = new ArrayList<String>();       // 右侧类型明确
var response = httpClient.send(request, handler); // 返回类型不直观时可考虑显式声明
```

- 右侧类型明显、显式类型只是重复时，`var` 通常能减少噪声。
- 类型表达重要业务含义，或者需要读者跳转才能知道类型时，使用显式类型。
- `var list = new ArrayList<>();` 会缺少目标类型并推断为 `ArrayList<Object>`，需要写出泛型参数。
- 使用 `var` 不会改变对象的运行时类型，也不会改变代理对象的行为。

## 2. 控制流：优先表达意图

- 需要产生一个值时，switch 表达式通常比“先声明变量、再在每个分支赋值”更清楚。
- 箭头分支不会穿透；只有确实需要穿透语义时才使用传统 `case:`。
- switch 表达式必须穷尽。对 enum 或 sealed 类型，可利用穷尽性检查；是否保留 `default` 取决于未知输入的处理策略。
- 模式匹配 switch、记录模式和 sealed 类型需要类型建模基础，统一在第 07 章学习。
- 多层嵌套影响阅读时，考虑提前返回、合并简单条件或抽取小方法。

## 3. 数值：先明确业务语义

- 整数运算可能静默溢出；需要检测时使用 `Math.addExact`、`Math.multiplyExact` 等方法。
- `double` 和 `float` 使用二进制浮点表示，不适合需要十进制精确结果的金额计算。
- 金额通常使用最小货币单位的整数，或使用明确精度与舍入规则的 `BigDecimal`。
- 不依赖复杂的运算符优先级；括号可以直接表达计算顺序。

## 4. 字符串：区分文本、字符和码点

- `String` 是 UTF-16 code unit 序列；`length()` 返回的不是必然等于用户看到的字符数。
- 内容比较使用 `equals`，引用身份比较才使用 `==`。
- 少量、直观的拼接使用 `+`；循环中逐步构建内容通常复用 `StringBuilder`。
- 多线程代码优先让构建器保持在线程内部，不要因为“多线程环境”就直接选择 `StringBuffer`。
- 空字符串与空白字符串含义不同，应按业务选择 `isEmpty()` 或 `isBlank()`，两者都不能直接处理 `null`。
- `strip()` 按 Unicode 空白规则处理，`trim()` 处理较早期的字符范围；根据输入协议选择，不必机械替换。

## 5. 文本块：改善多行文本可读性

文本块适合 SQL、JSON、HTML、帮助信息等多行静态内容。编译器会根据内容行和结束分隔符计算附带缩进，修改分隔符位置可能改变结果。

```java
String query = """
        SELECT id, name
        FROM users
        WHERE status = ?
        """;
```

文本块仍然只是字符串，不会自动转义 JSON、HTML 或 SQL。SQL 中的动态值应使用 `PreparedStatement` 等参数绑定机制，不要通过 `formatted()` 拼接不可信输入。

## 6. 数组与集合：按数据模型选择

- 长度固定、需要基本类型存储、底层 API 明确要求数组时，数组很合适。
- 需要动态增删、集合算法或清晰的只读语义时，通常选择集合。
- 数组索引范围始终是 `0` 到 `length - 1`；空数组和 `null` 数组是不同状态。
- 不要仅凭“数组更快”选择数据结构。先保证模型正确，性能敏感时再基于真实负载测量。

## 7. 循环：选择最直接的形式

- 需要索引、原地修改数组元素时使用下标 `for`。
- 只读取每个元素时优先增强 `for`。
- 迭代次数未知、由状态决定时使用 `while`。
- 标签可以直接退出多层循环，但复杂退出逻辑通常抽成方法更容易测试。
- Stream 是另一种数据处理抽象，不是循环的无条件替代品；在第 04 章学习后再按可读性和场景选择。

## 学习检查

完成本阶段后，应能解释：

1. `int` 为什么会溢出，`double` 为什么可能出现 `0.1 + 0.2 != 0.3`。
2. `==` 与 `equals` 在引用类型上的区别。
3. switch 语句与 switch 表达式的差异。
4. 数组的默认值、索引边界和引用语义。
5. `String`、`StringBuilder` 和文本块分别解决什么问题。
